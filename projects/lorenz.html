Lorenz Attractor

Introduction

The Lorenz system is a set of three ordinary differential equations, first developed by the meteorologist Edward Lorenz while studying atmospheric convection. It is a classic example of a system that can exhibit chaotic behavior, meaning its output can be highly sensitive to small changes in its starting conditions.

Overview

The behaviour of the position of the points is governed by the following equations: $$ {dx\over dt} = {\sigma(y-x)} $$ $$ {dy\over dt} = {x(\rho-z)-y} $$ $$ {dz\over dt} = {xy-\beta z} $$ Each point has a coordinate component which are calculated using equations above. The constants \(\sigma\), \(\rho\) and \(\beta\) (sigma, rho and beta respectively) are the same for every point. The points differ in their initial positions. From the beginning of the simulation, we see all the points following the same path, but as the simulation continues the points begin to diverge. The Lorenz Attractor is an example of a chaotic system, although it is deterministic; meaning if the simulation is run again we will get the same results. The results only change if the initial conditions are different.

The simulation above is run with the values \(\sigma=10\), \(\rho=28\) and \(\beta={8\over 3}\). The points start at position (1, 1, 1) and each other point is a difference of 0.05 units.

Visualisation

The position of each point is stored in a buffer each frame. Drawing the trail of each point is as simple as drawing a line between each point. The buffer has a size limit though, so after a certain period of time the simulation will stop.
